Projetando

O projetando é uma aplicação web desenvolvida por Maria Luiza Duarte, Lavínia Orth, Maria Paula Severo e Daniele Bernardes com o objetivo de auxiliar usuários no planejamento, organização e acompanhamento de seus projetos de forma simples e eficiente. A proposta central do sistema é fornecer uma interface intuitiva e funcional, que permita o controle total das tarefas envolvidas em cada projeto, desde a criação até a finalização.
